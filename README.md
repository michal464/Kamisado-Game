# Kamisado

משחק לוח דו-שחקני המבוסס על המשחק הקלאסי Kamisado, ממומש כאפליקציית Web צד-לקוח בלבד.

## תיאור המשחק

Kamisado הוא משחק אסטרטגיה על לוח 8×8 עם 8 צבעים. כל שחקן מזיז כלים קדימה (ישר או באלכסון) — הצבע של הריבוע שאליו הגעת קובע איזה כלי היריב יזיז בתור הבא. המנצח הוא השחקן הראשון שמגיע לשורת הבסיס של היריב.

## מבנה הפרויקט

```
kamisado/
├── html/         # דפי HTML (דף הבית, משחק, הוראות, רשימת משחקים, מנצח)
├── css/          # עיצוב לכל דף
├── js/           # לוגיקת המשחק וניהול משתמשים
├── images/       # תמונות ואייקונים
└── music/        # אפקטי קול (מהלך, שגיאה, ניצחון)
```

## טכנולוגיות

- HTML5, CSS3, Vanilla JavaScript
- `localStorage` / `sessionStorage` לניהול משתמשים ושמירת מצב

## הוראות הפעלה

1. פתח את `html/home-page.html` בדפדפן
2. הרשם או התחבר עם שני שחקנים
3. בחר משחק מרשימת המשחקים
4. שחק!

## כללי המשחק בקצרה

- שחקן שחור מתחיל ראשון — בוחר כל כלי שחור לצעד הראשון
- לאחר מכן, צבע הריבוע שנחתם קובע את הכלי הבא של היריב
- ניתן לנוע קדימה: ישר או באלכסון, ללא קפיצה מעל כלים
- מגיע לשורה הנגדית — מנצח!
